# Flight Booking System ✈️

A full-stack flight booking web application built using Node.js, Express, SQLite, and a responsive frontend. 
It allows users to search routes using Dijkstra’s algorithm, view available flights, select options, and book tickets with real-time status.
Booking history is stored and displayed dynamically. The project demonstrates backend API design, algorithmic routing, database integration, and modern UI/UX. 
Designed as a scalable mini version of platforms like MakeMyTrip for learning and portfolio purposes.
